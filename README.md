# HTML II Assignment Project

## Overview
This project is a simple two-page website built using HTML. It demonstrates the use of semantic HTML elements, forms, media elements, and navigation between pages.

## Pages Included
- **index.html** – Home page with personal information, media content, and embedded maps/videos
- **contact.html** – Contact page with a fully functional form

## Features

### Navigation
Both pages are connected using a navigation bar inside the `<header>` section.

### Semantic HTML
The project uses proper semantic elements such as:
- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<aside>`
- `<footer>`

### Media Elements
The project includes:
- Audio player
- Video player
- Embedded Google Map
- Embedded YouTube video

### Contact Form
The contact page includes a form with:
- Full Name input
- Email input
- Password input
- Radio buttons (Gender)
- Checkboxes (Hobbies)
- Dropdown menu (Country selection)
- Textarea (Message box)
- Submit button

## Technologies Used
- HTML5

## Purpose
This project was created as an assignment to demonstrate understanding of HTML structure, forms, and webpage navigation.
